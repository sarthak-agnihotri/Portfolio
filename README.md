# Sarthak Agnihotri - CS Portfolio

A modern, professional portfolio website designed for a Computer Science student.

## Folder Structure
```
Portfolio/
├── index.html   (Structure & Content)
├── style.css    (Design & Responsive Layout)
├── script.js    (Interactivity & Animations)
└── README.md    (This file)
```

## How to Run Locally
1. Clone or download this folder to your local machine.
2. Open the `index.html` file in any modern web browser (Chrome, Firefox, Safari, Edge).
3. Alternatively, if you use VS Code, you can use the **Live Server** extension to host it locally.

## Features
- **Modern Aesthetics**: Glassmorphism design and vibrant color palette.
- **Fully Responsive**: Works seamlessly on mobile, tablet, and desktop.
- **Micro-Animations**: Scroll reveal effects and smooth transitions.
- **Modular Sections**: Easy to add more skills or projects later.
- **Clean Code**: Well-commented HTML, CSS, and JS.
